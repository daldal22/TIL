# Markdown

일반 텍스트 기반의 경량 마크업 언어

서식이 있는 문서를 작성하는 데 사용되며, 일반 마크업 언어에 비해 문법이 쉽고 간단한 것이 특징이다.

## Header

* *누르고띄어쓰기
* #~######까지 h1~h6
  점있는 상태로 딜리트 누르면 들여쓰기 된 채로 쓸 수 있음

* **안녕** 글자 양옆에 **누르면 글씨 두꺼워짐

## Blockquote

* `>` , `>>`

  > 인용 문구
  >
  > > 중첩

## Code Block

* **```** / **~~~**

  코드 블럭
  첫 **```**옆에(뒤) 프로그래밍 언어를 작성할 수 있다.

  ```js
  var a = 123
  ```
  

## Inline Code Block

* **`**

  `Inline Code Block`

## Emphasi

* ***** / ******
  **_** / **__**
  강조를 위해 문자를 꾸밀 수 있다.
  *Italic*

  __Bold__

  **_Italic + Bold_**
